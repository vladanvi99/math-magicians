# Calculator App

> This is Calculator App.

![screenshot](./Screenshot_2.png)

Here you can calculate basic mathematical operations.

## Setup

### Clone this repository

```bash
$ git clone https://github.com/vladanvi99/math-magicians.git
$ cd math-magicians
```

### Run project

```bash
$ npm install
$ npm start
```
